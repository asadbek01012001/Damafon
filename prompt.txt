## 🧠 Project Goal
I have a Dahua intercom (video door phone). I want to integrate it into a web system so that:

- When someone presses the doorbell, a call is triggered via SIP
- Asterisk receives the SIP call from the Dahua device
- My backend (.NET or Node.js) receives call events from Asterisk (via ARI or AMI)
- A React web app shows an incoming call popup in real-time
- The user can:
  - Answer / Reject the call
  - View live video stream from the intercom
  - Speak with the visitor (two-way audio)
  - Open the door remotely

## 📡 Video Requirement
- Dahua provides RTSP stream
- The system must convert RTSP → WebRTC for browser playback with low latency
- FFmpeg or a media server (Janus / mediasoup / similar) can be used

## ☎️ Telephony Requirement
- Use Asterisk as SIP PBX
- Dahua device is a SIP client
- Backend must integrate with Asterisk using ARI (preferred) or AMI
- All call events must be pushed to frontend via WebSocket/SignalR

## 🧩 Backend Requirements
Design backend in either:
- ASP.NET Core 8 (preferred) OR Node.js (Express/NestJS)

Backend responsibilities:
- Handle Asterisk events (call started, answered, ended)
- Manage call sessions
- Trigger door unlock API on demand
- Stream control coordination
- WebSocket/SignalR real-time communication with frontend

## 🌐 Frontend Requirements
React.js application:

- Real-time incoming call popup
- Video player for live stream
- Buttons:
  - Accept call
  - Reject call
  - Open door
- Live audio/video communication support
- WebSocket/SignalR integration

## 🔐 Door Control
